🔑 Token de acceso GitHub

Para leer y guardar datos en GitHub necesitas un Personal Access Token. Se guarda solo en este navegador.

Cómo crear tu token (solo la primera vez):
1. Ve a github.com → Settings → Developer settings
2. Clic en Personal access tokens → Tokens (classic)
3. Clic en Generate new token (classic)
4. Nombre: actividad-diaria
5. Activa el permiso: repo
6. Clic en Generate token y copia el valor

Gerentes: usar permiso repo (solo lectura activa automáticamente)

El token se guarda únicamente en este navegador. Nadie más puede verlo.
